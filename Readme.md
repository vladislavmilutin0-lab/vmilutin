ЭПР — это радиотехническая характеристика объекта, которая показывает, насколько он «заметен» для радиолокационной станции (РЛС).
ЭПР позволяет:
•	Рассчитать максимальную дальность обнаружения РЛС.
•	Оценить уязвимость объекта для радиолокационного наблюдения.
•	Разрабатывать меры снижения заметности.
